# Instant Messaging App

A **real-time messaging applcation** designed for effortless communication. Features include:

- **Direct & Group Chats**: Connect with individuals or teams.
- **Reactions & Replies**: Express yourself and keep conversations organized.
- **Threaded Messages**: Dive deeper into specific topics without cluttering the main chat.
- **Video Calls**: Seamless integration for face-to-face conversations.
- **Real-Time Updates**: Instant message delivery and notifications.

---

## **Main Tech Stack**

| Tool          | Description                |
| ------------- | -------------------------- |
| **Next.js**   | Full-stack React framework |
| **Clerk**     | Secure user authentication |
| **Convex**    | Backend database           |
| **getStream** | Chat API infrastructure    |
